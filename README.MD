# Настройка облака

## Перед запуском:
- скопировать ssh ключ в файл с именем `ansible_key` в корень проекта для подключения к ВМ
- вписать корректный адрес машины в `hosts.ini` и `chaos.sh`
- при необходимости в файлах `hosts.ini` и `chaos.sh` поменять имя юзера под которым подключается ansible
- при отсутствии установить docker
- версия ubuntu на которой выполнялось тестовое: `24.04`
Для развертывания кластера надо выполнить команду
```shell
sh run-playbook.sh
```
Займет примерно 2-3 минуты зависит от RAM (должно быть 4ГБ минимум)

## Для проведения тестов

```shell
sh chaos.sh first-scenario.yaml
sh chaos.sh second-scenario.yaml
sh chaos.sh third-scenario.yaml
```
